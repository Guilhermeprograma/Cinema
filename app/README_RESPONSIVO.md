CINEPLUS - SISTEMA COMPLETAMENTE RESPONSIVO E FUNCIONAL
========================================================

📱 SISTEMA 100% RESPONSIVO E FUNCIONAL PARA TODOS OS DISPOSITIVOS

DESCRIÇÃO
---------
Sistema completo de reserva de cinema com interface moderna e totalmente responsiva.
Funciona perfeitamente em desktop, tablet e celular (iOS/Android).

ARQUIVOS INCLUÍDOS
------------------
- cineplus_mobile.html: Arquivo principal (HTML + CSS + JavaScript embutidos)
- index.html: Versão desktop
- style.css: Estilos separados
- script.js: JavaScript separado

COMO USAR NO CELULAR
--------------------

1. **TRANSFERIR O ARQUIVO**
   - Envie o arquivo ZIP para seu celular via WhatsApp, email ou transferência
   - Arquivo: CINEPLUS_RESPONSIVO_COMPLETO.zip

2. **EXTRAIR**
   - Abra o arquivo ZIP no celular
   - Extraia todos os arquivos em uma pasta
   - IMPORTANTE: Não abra diretamente no "preview" do gerenciador de arquivos

3. **ABRIR NO NAVEGADOR**
   - Use Chrome, Safari, Edge ou qualquer navegador moderno
   - Abra o arquivo: cineplus_mobile.html
   - O sistema carregará automaticamente
